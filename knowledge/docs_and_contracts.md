# Документы, договоры, заявки (kmipt.ru)

Последняя проверка: 2026-02-14.

## Базовый набор документов по учебным программам
- Договор на обучение.
- Платежные документы (чек/выписка/платежное поручение).
- Лицензия образовательной организации (копия по запросу).
- Документы о родстве с ребенком (для вычетов/оформления некоторых заявлений).

## Для налогового вычета (по материалам сайта)
- Копия договора на обучение.
- Копии лицензии образовательной организации.
- Копии платежных документов.
- Справка об очной форме обучения (если требуется по процедуре).
- Копия документа о родстве с ребенком.
- Справка о доходах и декларация 3-НДФЛ.
- Если платеж оформлен на второго родителя: свидетельство о браке.

## Для оплаты материнским капиталом (по материалам сайта)
- Скан сертификата на материнский капитал.
- Скан паспорта родителя (включая страницу с пропиской).
- Скан свидетельства о рождении обучающегося.
- Если маткапитал уже использовался: скан выписки об остатке средств.
- При несовпадении адресов: дополнительное указание адреса ребенка.
- Пакет документов отправляется на edu@kmipt.ru; далее организация готовит договор и комплект в Социальный фонд.

## По возврату
- Заявление на возврат.
- Чек об оплате.
- Реквизиты карты, с которой была оплата.

## Что бот должен делать по документам
- Давать только список документов и следующий шаг (куда отправить, с кем связаться).
- Если клиент просит юридическое толкование договора/условий, переводить на менеджера.
- Не обещать результат по вычету/маткапиталу до решения ФНС/СФР.
